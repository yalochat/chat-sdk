# Yalo Chat SDK

A cross-platform UI package for easy integration of Android, iOS, and web applications with Yalo's chat services.

## Features

- Cross-platform support (Android, iOS, Web), via our Flutter and future native packages
- Easy integration with Yalo services
- Pre-built UI components
- Customizable chat interface
